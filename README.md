## 🔹 Project Overview
This project focuses on analyzing healthcare data to understand patient trends, hospital performance, and financial metrics. The aim is to generate insights that support better decision-making and improve operational efficiency.

## 🔹 Tools Used
- SQL  
- Excel  
- Power BI  
- Tableau  

## 🔹 Key Metrics
- Total Patients  
- Total Visits  
- Revenue  
- Doctor Workload  
- Treatment Cost  

## 🔹 Key Insights
- Identified patterns in patient visits and treatment trends  
- Analyzed doctor workload to improve resource allocation  
- Evaluated costs and revenue for better financial planning  

## 🔹 Conclusion
This project demonstrates how data analysis and dashboards can help in improving healthcare operations and decision-making.
